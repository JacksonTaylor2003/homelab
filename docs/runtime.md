# Runtime Conventions

## Base paths

Docker service data should live under:

```text
/opt/homelab
```
stack data follows this pattern:

```text
/opt/homelab/<stack-name>
```

examples:

```text
/opt/homelab/media
/opt/homelab/seafile
/opt/homelab/vikunja
/opt/homelab/minecraft
/opt/homelab/gitea
/opt/homelab/homer
/opt/homelab/monitoring
```
